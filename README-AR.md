## كيفية الاستخدام
1) ارفع محتويات هذا المجلّد إلى مستودعك `SAT-EXAM` على GitHub.
2) فعّل GitHub Pages من إعدادات المستودع (Settings → Pages → Branch: `main` → `/root`).
3) افتح الرابط: `https://<username>.github.io/SAT-EXAM/`.
